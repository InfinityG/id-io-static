What is the ID-IO client?

The ID-IO client is a browser app that uses ID-IO API to manage your identity. It demonstrates the following:

  • Registration on the blockchain using the ID-IO API
  • Stores your signing keys in an offline wallet
  • Create connections to other ID-IO users
  • Logging into a relying 3rd party website using the ID-IO SSO (single-sign-on) mechanism

Not registered?

Registration will create a new identity for you on the blockchain. It will also create an encrypted offline wallet in your browser storage to keep your keys safe.
